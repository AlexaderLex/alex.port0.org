Командная строка всегда была незаменимым инструментом системного администратора, однако простым пользователям она обычно представляется чем-то сложным, непонятным и ненужным.

Тем не менее есть немало команд Windows которые пригодятся не только гикам и сисадминам, но и обычным пользователям этой операционной системы. Предлагаю Вашему вниманию 10 команд Windows полезных каждому.

Чтобы выполнить эти команды нажимаем сочетание клавиш Win + R и вводим нужную команду или можно создать ярлык, а в поле «укажите расположение объекта» ввести нужную команду.

  • control — вызывает классическую панель управления. Удобно в всете того, что Microsoft в каждой последующей версии Windows всё глубже прячем классический интерфейс.
  • control userpasswords2 — ещё одно окно наследия Windows 2000. Очень удобное и компактное.
  • rundll32 user32,LockWorkStation — с помощью этой команды можно создать ярлык для блокировки ПК
  • explorer ::{2227a280-3aea-1069-a2de-08002b30309d} — открыть классические принтеры и факсы. У этой команды есть алиас explorer shell:printersfolder
  • explorer ::{208d2c60-3aea-1069-a2d7-08002b30309d} — открыть старое сетевое окружение, где есть значок «вся сеть»
  • timedate.cpl — открыть удобное окно управления временем
  • ncpa.cpl — открывает окно, в котором видны все сетевые интерфейсы
  • desk.cpl ,,5 — выводит окно управления значками на рабочем столе. (оказывается работает и вариант с одной запятой desk.cpl ,5)
  • rasphone — удобная программа управления подключениями удалённого доступа
  • appwiz.cpl — классическое окошко программы и компоненты

 

Как вы уже догадались, эти команды командной строки Windows (cmd.exe) позволяют обратиться к встроенным функциям ОС, при этом вызвать команду зачастую быстрее, чем найти куда в очередной раз запрятали ту или иную настройку.
