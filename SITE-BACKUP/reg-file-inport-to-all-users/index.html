REG Importer. Импорт файлов реестра для всех пользователей cats: AutoIT, Windows, Программы, Сисадминство tags: AutoIT, windows, Командная строка, Программы 2018-07-13 15-13

Сисадмины, авторы репаков и сборок Windows, а также просто любители твиков и автоматизации часто сталкиваются с необходимостью внести изменения в реестре сразу для всех пользователей, а также в профиль пользователя Default. Для этих целей мною был написан скрипт на AutoIT.

Получилась очень удобная программка, которая принимает в качестве параметров имена .REG файлов и применяет их ко всем пользователям. При вызове без параметров запрашивает файлы реестра для импорта.

Исходный код прилагается.

REG_importer
