OpenOffice LibreOffice системная ошибка api-ms-win-crt-runtime-l1-1-0.dll cats: Прочее tags: 2017-11-07 14-56 libreoffice-system-error-api-ms-win-crt-runtime-l1-1-0-dll\

OpenOffice или LibreOffice не запускается. Пишет "Системная ошибка. Запуск программы невозможен так как на компьютере отсутствует api-ms-win-crt-runtime-l1-1-0.dll. Попробуйте переустановить программу."

Чтобы исправить эту ошибку, нужно установить Update for Universal C Runtime in Windows (или как оно называется в русской версии Обновление для универсальной среды выполнения C в Windows).

При этом Windows может быть и пиратским. Главное, чтобы не была отключена служба обновления, иначе установить этот пакет не получится.
