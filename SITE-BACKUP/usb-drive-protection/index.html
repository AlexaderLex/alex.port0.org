В 2015-м году вредоносных программ, использующих механизм автозапуска Windows уже не так много как раньше. В Windows Vista и последующих Microsoft закрыл серьёзную брешь в безопасности системы, отключив автозапуск для флешек. Тем не менее многие до сих пор работают с Windows XP для них эта статья будет актуальна.

Суть в том, чтобы не дать заражённому компьютеру записать на флешку файл autorun.inf

  1. посмотрите какая буква у вашей флешки в окне Компьютер
  2. Запустите командную строку Windows
  3. Введите команду md "\\?\E:\autorun.inf\nul"
    где E: – буква вашей флешки

Если команда выдала ошибку, то введите по очереди следующие команды:

del /f /q E:\autorun.inf
rd /s /q "\\?\E:\autorun.inf"

после этого команда под номером 3 должна отработать без ошибок, а в корне флешки появится папка autorun.inf, которую нельзя удалить. Если всё же нужно будет её удалить, используйте команду rd так, как она приведена выше.

Я подготовил архив usb-drive-protection с командными файлами, автоматизирующими этот процесс. Запускать файлы нужно непосредственно с защищаемой флешки. Это важно!

В архиве также есть файл, который препятствует созданию точек восстановления на внешних дисках. Полезно, если у вас есть внешний usb диск, который вы подключаете к разным компьютерам
